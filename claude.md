# プロジェクト概要

これは、X（旧Twitter）でテック系の情報発信をしつつ、たまにアフィリエイトリンク付きの投稿も行うための自動投稿ボットのリポジトリです。

## アカウントの方針

- テック系の話題を中心に発信する専門アカウントとする
- 毎日〜ほぼ毎日、テックに関する普通の投稿を行う
- アフィリエイトリンク付きの投稿は「たまに」に抑え、過剰な宣伝にはしない
- アフィ投稿時は #PR などの表記を入れて、広告であることを明示する

## ボットの役割

- テック関連ニュースやトピックから1件を選び、日本語で分かりやすいポスト文を生成する
- 必要に応じて、あらかじめ用意されたアフィリエイトリンクを1つだけ末尾に追加する
- アフィ投稿の頻度は、全体の1〜2割程度に収まるように調整する（設定値で変更可能にする）

## Claude Code にお願いしたいこと

- GitHub Actions ワークフローとスクリプト群を理解・整理する
- 「通常のテック投稿」と「アフィリエイト投稿」の2種類を切り替える設計に改善する
- 次の点を意識してコードを提案・修正してください:
  - 設定値（投稿頻度、アフィ投稿比率、使用するASPの種類など）は .env や GitHub Secrets に切り出す
  - 可読性を優先し、複雑なロジックは関数に分割する
  - 日本語コメントで「何をしているか」をざっくり説明する

## 開発ルール

- 言語は TypeScript / JavaScript / Python のうち、既存コードに合わせる
- 既存スタイルに合わせて、ESLint + Prettier を尊重する（設定ファイルがあればそれに従う）
- 動作確認用の簡単なテストも提案してほしい（例: dry_run モードでの実行）
